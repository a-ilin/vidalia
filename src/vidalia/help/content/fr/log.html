Journal des messages

Le Journal permet de suivre les informations d'état d'une instance de Tor en cours d'utilisation. Chaque message est associé a un niveau de gravité, depuis Erreur (le plus sérieux) jusqu'à Débogage (le plus bavard). Consultez la section suivante sur la gravité des messages pour davantage d'informations.

Niveaux de gravité des messages

Le niveau de gravité permet de connaître l'importance d'un message. Un message de gravité supérieure indique généralement qu'un problème est survenu avec Tor. Les messages de gravité inférieure sont fréquents durant les opérations normales de Tor et il est habituellement inutile de les enregistrer.

Les niveaux de gravité sont les suivants, du plus au moins sévère:

  • Erreur : Ces messages apparaissent lors des erreurs fatales, lorsque Tor ne peut pas poursuivre. Ces messages sont surlignés en rouge dans le Journal.
  • Avertissement : Ces messages s'affichent lorsque Tor rencontre un problème qui ne l'empêche pas de continuer. Ils sont surlignés en jaune.
  • Signalement : Ces messages apparaissent quelquefois lors des opérations normales de Tor. Ils ne sont pas considérés comme des erreurs, mais vous devez cependant y prêter attention.
  • Information : Ces messages renvoient aux opérations normales de Tor et ne présentent généralement pas d'intérêt pour la plupart des utilisateurs.
  • Débogage : Ces messages extrêmement complets sont avant tout destinés aux développeurs. Vous ne devez généralement pas les afficher, sauf si vous savez exactement ce que vous faîtes.

La majorité des utilisateurs doivent afficher seulement les messages d'erreur, d'avertissement et de signalement.

Pour choisir les niveaux de gravité des messages que vous souhaitez afficher, vous devez suivre cette procédure :

  1. Ouvrez le Journal des messages à partir du menu Vidalia situé dans la zone de notification.
  2. Cliquez Options en haut de la fenêtre du Journal.
  3. Cochez les niveaux de gravité que vous souhaitez afficher dans la fenêtre du Journal des messages et décochez les autres.
  4. Cliquez Enregistrer les options pour utiliser votre nouveau filtre des messages.

Enregistrer les messages dans un fichier

Vidalia peut également enregistrer le Journal des messages dans un fichier en même temps qu'il les affiche dans la fenêtre du Journal. Pour permettre cet enregistrement, suivez ces étapes:

  1. Ouvrez le Journal des messages à partir du menu Vidalia situé dans la zone de notification.
  2. Cliquez Options en haut de la fenêtre du Journal.
  3. Cochez la case intitulé Enregistrer automatiquement les nouveaux messages dans un fichier .
  4. Si vous voulez changer le fichier dans lequel les messages sont enregistrés, entrez le chemin et le nom du fichier dans le champ ou cliquez Parcourir pour naviguer jusqu'à votre fichier d'historique.
  5. Cliquez Enregistrer les options pour sauvegarder le chemin du Journal.
